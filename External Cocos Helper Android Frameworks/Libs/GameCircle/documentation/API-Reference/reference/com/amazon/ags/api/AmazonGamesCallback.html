public interface

AmazonGamesCallback

com.amazon.ags.api.AmazonGamesCallback

Class Overview


This interface is still under development and the API is subject to change or removal at any time without notice.
The API may change on your device if it is flashed with a new build or receives an OTA update.

Callback interface to handle all asynchronous responses from the Amazon Games. Implement this interface to override the default out-of-the-box behavior.
