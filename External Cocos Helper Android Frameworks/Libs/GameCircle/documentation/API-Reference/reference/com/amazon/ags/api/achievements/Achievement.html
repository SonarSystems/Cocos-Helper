public interface

Achievement

com.amazon.ags.api.achievements.Achievement

Class Overview


This interface is still under development and the API is subject to change or removal at any time without notice.
The API may change on your device if it is flashed with a new build or receives an OTA update.

Represents a single achievement. This is an immutable resource object which must be loaded with an additional call
