public interface

WhispersyncClient

com.amazon.ags.api.whispersync.WhispersyncClient

Class Overview


This interface is still under development and the API is subject to change or removal at any time without notice.
The API may change on your device if it is flashed with a new build or receives an OTA update.

Interface for accessing Whispersync for Games. Whispersync data is always accessible, even when the player is not registered with Amazon or is offline. Unlike the LeaderboardsClient and AchievementsClient, WhispersyncClient can be used immediately after calling AmazonGamesClient.initialize().

public abstract void migrateVersion1GameData (MigrationCallback callback)

For games that have been previously released on the Amazon Appstore and used the 1.x version of GameCircle Whispersync API, this method may be used to download existing game save data from the cloud. Typically, this is used to ensure a smooth upgrade experience for gamers who have previously synchronized game progress from a Kindle Fire. Consider the following scenarios: Scenario 1) A player has previously synchronized game progress on a Kindle Fire and has just upgraded the game on a generic Android device. After converting existing local game data to use GameCircle Whispersync, the game should use this method to download older Kindle Fire data and convert it too. Scenario 2) A player has previously synchronized game progress on a Kindle Fire, but has chosen to remove the game from their local library. Since it is a clean install, there is no local data to convert, however, the older version 1.x game data should be downloaded and converted to the current GameCircle Whispersync API. This method will attempt to access network resources, so do not call from main UI thread. Note: This method *only* applies to games that have integrated with GameCircle Whispersync for the Kindle Fire.

callback MigrationCallback to asynchronously return the result
Throws
NullPointerException if callback is null

public abstract void unpackVersion1MultiFileGameData (byte[] gameData, File overrideDir)

Unpacks and overlays multi-file game data that was received from calling migrateVersion1GameData(). Note: This method is intended only for games that have been previously released on the Amazon Appstore and used multi-file Whispersync from version 1.x of the GameCircle API.

Parameters
gameData byte[] received from MigrationCallback.onComplete()
overrideDir An optional directory File to specify were to unpack the game data. If null is specified, the default package directory used by version 1.x will be used. i.e. context.getFilesDir().getParentFile()
Throws
IOException in the event of failure while writing unpacked files to the data storage
