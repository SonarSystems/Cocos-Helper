public interface

SyncableNumberList

com.amazon.ags.api.whispersync.model.SyncableNumberList

Class Overview


This interface is still under development and the API is subject to change or removal at any time without notice.
The API may change on your device if it is flashed with a new build or receives an OTA update.

A syncable type that represents a list of fixed numbers. The list has a configurable max size, and is typically used to represent a top N list of highest numbers or lowest numbers across multiple devices. Examples are top 10 scores and top 10 fastest times. Note that metadata cannot be associated with this type, but metadata can be associated with its elements.

public abstract SyncableNumberElement[] getValues ()

Returns an immutable copy of the elements of this SyncableNumberList as an array. The size of the array is the lesser of its maxSize value and the total number of values added to the list.

Returns
  • an array containing up to maxSize number of elements in this SyncableNumberList.

public abstract boolean isSet ()

Returns whether or not this SyncableNumberList has been set in the GameDataMap. A SyncableNumberList has not been set, if and only if it is empty.

Returns
  • whether or not this SyncableNumberList has been set in the GameDataMap.

public abstract void setMaxSize (int size)

Sets the max size of this SyncableNumberList. size must be positive and no greater than MAX_SIZE_LIMIT. If size is smaller than the current size of this SyncableNumberList in the cloud, then its size will remain. In other words, the size of SyncableNumberList can never shrink.

Parameters
size the max size of this SyncableNumberList
Throws
IllegalArgumentException if size is non-positive or greater than MAX_SIZE_LIMIT.
