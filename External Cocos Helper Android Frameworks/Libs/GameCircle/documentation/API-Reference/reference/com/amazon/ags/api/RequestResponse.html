Known Indirect Subclasses
GetAchievementResponse, GetAchievementsResponse, GetLeaderboardPercentilesResponse, GetLeaderboardsResponse, GetPlayerScoreResponse, GetScoresResponse, RequestFriendIdsResponse, RequestFriendsResponse, RequestPlayerResponse, SubmitScoreResponse, UpdateProgressResponse
GetAchievementResponse The service response to a call to AchievementsClient.requestAchievement() Includes the Achievement asked for and the information encapsulated in the generic RequestResponse  
GetAchievementsResponse The service response to a call to AchievementsClient.requestAchievements() Includes the list of achievements for the game and the information encapsulated in the generic RequestResponse  
GetLeaderboardPercentilesResponse Response object for getting leaderboard percentiles for a leaderboard. 
GetLeaderboardsResponse A response to a request to get all leaderboards for a game Encapsulates the lists of uniqueID's and names of the leaderboards  
GetPlayerScoreResponse The response object returned after a call to RequestPlayerScore Extends RequestResponse and encapsulates the data requested and information about the results of the request  
GetScoresResponse A response to a request to get scores from a leaderboard Encapsulates a list of RankedScores along with meta-information about the leaderboard they came from  
RequestFriendIdsResponse The service response to a call to PlayerClient.getFriendIds() Includes a list of the player ID's for the local player's friends and the information encapsulated in the generic RequestResponse. 
RequestFriendsResponse The service response to a call to PlayerClient.getBatchFriends(...) Includes the profiles (as Player objects) of the requested players who are currently friends of the local player. 
RequestPlayerResponse The service response to a call to PlayerClient.requestLocalPlayer() Includes the requested Player object and the information encapsulated in the generic RequestResponse  
SubmitScoreResponse A response to a request to submit a score to a leaderboard. 
UpdateProgressResponse The service response to a call to AchievementsClient.updateProgress() Includes unlocked status of the achievement and the information encapsulated in the generic RequestResponse  

Class Overview


This interface is still under development and the API is subject to change or removal at any time without notice.
The API may change on your device if it is flashed with a new build or receives an OTA update.

RequestResponse make up the AmazonGamesEvents Queue Base class for all AmazonGames asset RequestResponses Can handle these by checking the event type and then casting
