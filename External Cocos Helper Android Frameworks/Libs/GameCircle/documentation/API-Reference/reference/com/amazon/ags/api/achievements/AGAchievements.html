public static void getAchievements (AGResponseCallback<GetAchievementsResponse> callback, Object... userData)

Get the list of all visible achievements for the game. Visible achievements include those that were never hidden, and those that were once hidden but since made visible. Each Achievement object in the list includes the current players progress toward the Achievement. The passed callback will be called when the operation is complete. This is an asynchronous operation.

public static void updateProgress (String achievementId, float percentComplete, AGResponseCallback<UpdateProgressResponse> callback, Object... userData)

Updates progress toward the specified achievement by the specified amount. If the submitted value is outside of range, it will be clamped to the 0 to 100 range inclusive. If the submitted value is less than the stored value, the update is ignored. The passed callback will be called when the operation is complete. This is an asynchronous operation.

Parameters
achievementId Achievement ID to update process on.
percentComplete Floating point value between 0.0f and 100.0f
callback Callback to call when the operation is complete.
userData Optional user data. Delivered to the user as an array of objects.
