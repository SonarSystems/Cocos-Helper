public interface

PlayerClient

com.amazon.ags.api.player.PlayerClient

Class Overview


This interface is still under development and the API is subject to change or removal at any time without notice.
The API may change on your device if it is flashed with a new build or receives an OTA update.

Interface for retrieving player information for a given player from the Amazon Game Service. There are two mechanisms for processing requests through this client: 1. Callbacks - A callback can be provided to process a response once the request is complete. 2. Handles - A handle is returned and can be used to check request status, and retrieve the response.

public abstract AGResponseHandle<RequestFriendsResponse> getBatchFriends (List<String> playerIds, Object... userData)

Asynchronously returns the player objects for the requested friends of the local player. Note: Among the playerIds supplied by the caller, only current friends of the local player will return player objects. All others will yield no results. An UNRECOVERABLE will be returned if the player is not authenticated, or the player is hidden.

Parameters
userData User data for the call.
Returns
  • Handle used to check request status and retrieve the response.
